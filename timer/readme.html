This is a very simple timer.

  • At 1 min the screen turns yellow
  • At 0 min the screen turns red
  • Time keeps running after hitting 0
You can control some of these values with URL params:
  • durationSecs - total length of the timer in seconds, defaults to 300 (5m)
  • warningAtSecs - when to show the warning color (yellow) in seconds, defaults to 60 (1m)
  • tickMillis - the number of millis in a tick, defaults to 1000 (1s). If you want to make the timer go faster decrease this number. The name is horrible, I know, because I'm a shitty namer.
Key shortcuts:
  • space - toggle
  • r - reset
